# StudyTrack

StudyTrack is a responsive student assignment management dashboard built with HTML, CSS, and JavaScript.

## Overview

StudyTrack helps students organize assignments, track deadlines, monitor progress, and manage academic tasks efficiently.

## Features

- Add assignments
- Delete assignments
- Mark assignments as completed
- Filter assignments by status

  All
  Pending
  Completed

- Assignment priority levels

  High
  Medium
  Low

- Due date tracking
- Completion rate analytics
- Statistics dashboard
- Local Storage persistence
- Responsive design

## Technologies Used

- HTML
- CSS
- JavaScript
- Local Storage API
- Git
- GitHub

## How to Run

1. Clone the repository

```bash
git clone https://github.com/sadik003/studytrack.git
```

2. Open the project folder in VS Code.

3. Open `index.html` using Live Server.
